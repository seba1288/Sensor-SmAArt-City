## 🚀 Quick Start (Local Development)

### 1. Clone Repository

```bash
git clone <your-repo-url>
cd Sensor-SmAArt-City
```

### 2. Frontend Setup (React/Vite)

Install the required Node dependencies and start the development server:

```bash
# Install dependencies
npm install

# Start the frontend app
npm run dev
```

The frontend application will be available at `http://localhost:8080/` (or the port specified in your terminal).
